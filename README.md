## Usage

Install this theme into `devDependencies`:

```bash
$ pnpm i dumi-theme-chaos -D
```


## Contributing

```bash
# not found antd dumi execute it
# pnpm add antd dumi --save-peer

# install deps
pnpm install

# start dumi server
pnpm run start
```
